# Apache Superset Sandbox

## Introduction
1. Change directory to `superset`
    ```shell
    cd superset 
    ```
2. Run the following command to start the Apache Superset service
    ```shell
    docker compose up -d
    ```
3. Access http://localhost:8088/
4. Login with `admin`/`admin`
